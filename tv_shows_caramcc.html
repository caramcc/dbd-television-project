TV Shows (caramcc)

These tables contain data about television shows. Information about a given television show that can be found in these tables are:

  • Title - show_title
  • Country of origin - country
  • Date the first episode aired - start_date
  • Date the last episode aired - end_date
  • FCC content rating (american shows only) - content_rating
  • Show category (e.g. scripted, reality, talk show) - classification
  • How long an episode lasts - runtime
  • The network which produced it - network_name
  • What time of day it aired (local to where it aired) - airtime
  • What timezone its airtime is reported in - timezone
  • Summary of the show's plot - plot_summary
  • Number of awards the show was nominated for - award_nominations
  • Number of awards the show won - award_wins
  • Overall rating on IMDB - imdb_rating
  • Number of votes on IMDB - imdb_votes
  • The show's ID on IMDB - imdb_id
  • The show's ID on TVRage API - tvrage_id
  • The show's official twitter handle, if applicable - show_twitter_id
  • The languages the show was broadcast in - show_languages table
  • The genres the show belongs to - show_genres table
  • The days of the week the show aired on - show_airdays table
  • Other titles the show is known by - show_alternate_titles table
  • The names and twitter handles of actors who appeared in the show - actors table
  • The names and twitter handles of the show's creators - creators table
